# Source Material

Drop files here that Claude should ingest to understand your voice, philosophy, and teaching style.

## What goes here
- Rebuildr Certification Course slide decks (PDF, PPTX, or Google Slides exports)
- Any other educational content you've created
- Brand guidelines, tone references, etc.

## How to add
Just drop files into this folder and push:
```bash
cd ~/Documents/Claude-Work/claude-memory
git add source-material/
git commit -m "Add source material"
git push
```

Or just drop them in and the hourly sync will pick up any .md files automatically.
For non-.md files (PDFs, PPTX), push manually with the commands above.
