---
name: UI spacing and density preferences
description: Tayla finds dense/crowded text overwhelming — needs generous spacing, bigger buttons, more padding on pills and badges
type: feedback
---

Text on pages looks "way too crowded" — needs more breathing room between elements.
Background colors on buttons/text need to be "expanded slightly" — more padding on pills, badges, buttons.

**Why:** Tayla is a visual person who values clean, minimal design. Dense text feels overwhelming.
**How to apply:** Default to generous spacing (py-8+ between sections, gap-4+ between items, px-5+ py-2+ on pills). Err on the side of too much whitespace rather than too little. Buttons should feel substantial, not cramped.
